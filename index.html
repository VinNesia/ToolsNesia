Cara Menggunakan Tools

  • Langkah-langkah: Pilih tanggal lahir di kolom "Usia" dan klik "Hitung Usia" untuk melihat usia Anda. Untuk BMI, masukkan berat badan (kg) dan tinggi badan (cm), lalu klik "Hitung BMI".
  • Contoh: Jika tanggal lahir 01/01/2000, hasilnya akan menunjukkan usia Anda hari ini. Untuk BMI, masukkan 70 kg dan 170 cm, hasilnya sekitar 24.22.
  • Tips: Pastikan input valid (tidak negatif) agar tidak ada error.
  • Langkah-langkah: Ketik URL (misalnya https://example.com) atau teks di kolom input, lalu klik "Buat QR" untuk menghasilkan kode QR.
  • Contoh: Masukkan "https://vinnesia.github.io", lalu klik tombol untuk melihat kode QR-nya.
  • Tips: Gunakan URL yang valid untuk hasil terbaik. Kode QR bisa discan dengan aplikasi kamera HP.
  • Langkah-langkah: Klik tombol "Buat Password" untuk menghasilkan password acak berisi 12 karakter (huruf, angka, dan simbol).
  • Contoh: Hasil mungkin seperti "Kj9$mPq2@xL4".
  • Tips: Salin password segera setelah dibuat untuk keamanan.
  • Langkah-langkah: Masukkan jam, menit, dan detik (misalnya 0, 0, 10 untuk 10 detik), lalu klik "Mulai" untuk memulai penghitung waktu.
  • Contoh: Set 0 jam, 1 menit, 0 detik, lalu klik "Mulai" untuk hitung mundur dari 1:00 ke 0:00.
  • Tips: Pastikan nilai tidak negatif, dan tekan "Kembali ke Beranda" untuk reset.
  • Langkah-langkah: Ketik teks di area teks (misalnya "hello world"), lalu pilih "Huruf Besar", "Huruf Kecil", atau "Kata Besar" untuk mengubah format.
  • Contoh: "hello world" menjadi "HELLO WORLD" (Huruf Besar), "hello world" (Huruf Kecil), atau "Hello World" (Kata Besar).
  • Tips: Gunakan untuk format teks dokumen atau caption media sosial.
  • Langkah-langkah: Masukkan jumlah (misalnya 100), pilih mata uang asal (USD atau IDR) dan tujuan, lalu klik "Konversi".
  • Contoh: 100 USD ke IDR dengan rate 14,500 akan menghasilkan 1,450,000 IDR.
  • Tips: Rate adalah contoh (1 USD = 14,500 IDR), hasil hanya estimasi.
